# OOCalculator
Sample app for teaching GitFlow

## Requirements
- dotnetcore sdk 2.0

## How to run
- git clone <this respository url>
- cd OOCalculator
- dotnet restore
- dotnet build
- dotnet run -p .\src\OOCalculator

## Exercise
- New commands like "-", "*", "/", "%" and other.
- Add unit tests
- Red error message
- Introduce IoC Container
